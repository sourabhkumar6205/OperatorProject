# Operator Project (JavaScript Calculator)

This project is a basic calculator built with HTML, CSS and JavaScript.
It performs operations like addition, subtraction, multiplication, division and modulus.
The layout includes keyboard navigation support so users can move through the form using arrow keys and Enter.

---

## Features

### ✔ Perform Basic Operations

* Addition
* Subtraction
* Multiplication
* Division
* Modulus

### ✔ Keyboard Navigation

* Press Enter to jump between input fields
* Use Arrow keys to move between operator buttons
* Smooth focus control for better accessibility

### ✔ Clean UI

* Simple design
* Highlighted buttons
* Clear result display
* Auto-clear input fields after each calculation

---

## How It Works

1. Enter two numbers in the input fields.
2. Click any operator button or navigate using arrow keys.
3. The result is shown in the Result box.
4. After each calculation, the number fields reset automatically.
